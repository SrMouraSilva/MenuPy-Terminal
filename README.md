MenuPy-Terminal
===============

A creator of menus for terminal customization options

Sobre
=====
Sistema gerador de menu para prompt de comando.

Versão atual
============
Versão Beta 0.9.0

Wiki
====
https://github.com/SrMouraSilva/MenuPy-Terminal/wiki

Características
===============

- Multiplataforma - Windows e Sistemas Baseados em Unix
- Fácil utilização
- Exportável para XML (implementando)
- Visual customizável
- Tratamento de erros (implementando)
- Internacionalização do código
- Customização: Texto colorido
